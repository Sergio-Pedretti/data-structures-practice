## Number of binary trees

### Catalan Number:

$$T(n) = {C^{2n}_n \over (n+1)}$$

#### or could be described as:

$$T(n) = {\sum_{i=0}^{n} {T(i)*T(n-i)}}$$

### Also, if all nodes are labelled, so position could change:

$$T(n) = {C^{2n}_n \over (n+1)}*{n!}$$

### Max Height:

$$T(n) = {2^{n-1}}$$
